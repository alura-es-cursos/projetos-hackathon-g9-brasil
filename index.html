Contexto

Sobre o Hackathon

Uma maratona de inovação para os alunos do programa ONE colocarem em prática os conhecimentos adquiridos, desenvolvendo soluções reais com tecnologia e inteligência artificial.

🎯

Objetivo

Desenvolver MVPs funcionais que integrem Ciência de Dados, APIs REST e serviços de nuvem OCI, solucionando problemas reais de mercado.

👥

Equipes

Os alunos se organizam em grupos multidisciplinares na plataforma NoCountry, unindo perfis de Back-End, Ciência de Dados e mais.

☁️

Tecnologias

Python, Scikit-Learn, Java Spring Boot e Oracle Cloud Infrastructure (OCI) são as principais tecnologias envolvidas nos projetos.

🏆

Entregas

Cada equipe entrega um notebook de Data Science, uma API REST documentada e integração com ao menos um serviço OCI.

01
🧠
NLP · Gestão do Conhecimento

ContentAI

Classificação e Organização Inteligente de Conteúdos

Descrição do Projeto

Criar uma solução inteligente capaz de receber conteúdos técnicos em formato textual e processá-los automaticamente, classificando-os em categorias relevantes, extraindo palavras-chave e enriquecendo as informações com metadados gerados pelo modelo.

O objetivo é facilitar a organização e o acesso a grandes volumes de informação, tornando repositórios de conhecimento mais eficientes e navegáveis.

🏢 Setor

Plataformas educacionais, documentação técnica e gestão do conhecimento corporativo.

Stack Técnica

Python Pandas Scikit-Learn TF-IDF Java Spring Boot OCI

📦 Entregas Obrigatórias (MVP)

  • Notebook com EDA, treinamento e serialização do modelo
  • API REST com endpoint POST /conteudo
  • Retorno JSON com categoria, probabilidade e palavras-chave
  • Integração com ao menos um serviço OCI
  • Mínimo de 3 exemplos de uso documentados

⚡ Exemplo de Requisição

{
  "titulo": "Introdução ao Spring Boot",
  "texto": "Conceitos básicos para
  criação de APIs REST com
  Java e Spring Boot."
}

✅ Exemplo de Resposta

{
  "categoria": "Backend",
  "probabilidade": 0.89,
  "informacoes_adicionais": [
    "Java", "Spring Boot",
    "API REST"
  ]
}

🚀 Recursos Opcionais

  • Recomendação de conteúdos relacionados
  • Busca semântica
  • Dashboard simples
  • Processamento em lote (CSV)
  • Containerização com Docker
OCI obrigatório: Object Storage · Compute · Functions

02
📊
Fintech · Educação Financeira

FinanceAI

Assistente Inteligente de Saúde Financeira

Descrição do Projeto

Criar uma solução inteligente capaz de analisar o comportamento financeiro de um usuário a partir de suas transações, gerando uma visão mais completa da sua saúde financeira.

A solução classifica despesas automaticamente, identifica padrões de consumo, define o perfil financeiro do usuário e apresenta recomendações personalizadas para melhorar sua organização financeira.

🏢 Setor

Fintech, bancos digitais, carteiras digitais e plataformas de educação financeira.

Stack Técnica

Python Pandas Scikit-Learn Java Spring Boot OCI

📦 Entregas Obrigatórias (MVP)

  • Notebook com EDA, engenharia de atributos e modelos
  • API REST com endpoints de classificação de despesas
  • Classificação de perfil financeiro (ex: Organizado, Endividado)
  • Recomendações personalizadas em JSON
  • Alertas de gastos elevados
  • Integração com ao menos um serviço OCI

⚡ Exemplo de Requisição

{
  "descricao": "Uber",
  "valor": 45.00,
  "renda_mensal": 3500,
  "frequencia_economia": "baixa",
  "nivel_endividamento": "moderado"
}

✅ Exemplo de Resposta

{
  "categoria_despesa": "Transporte",
  "perfil_financeiro": "Moderado",
  "recomendacoes": [
    "Considere alternativas de
    transporte mais econômicas",
    "Estabeleça um orçamento
    mensal de transporte"
  ]
}

🚀 Recursos Opcionais

  • Dashboard de acompanhamento
  • Histórico de análises
  • Exportação de relatórios
  • Processamento em lote via CSV
  • Containerização com Docker
OCI obrigatório: Object Storage · Compute · Functions

03
⚡
Sustentabilidade · Energia

EnergiAI

Inteligência para Consumo Energético

Descrição do Projeto

Criar uma solução inteligente capaz de analisar padrões de consumo de energia elétrica e gerar informações que auxiliem na tomada de decisões relacionadas à eficiência energética.

A aplicação utiliza Ciência de Dados para identificar padrões e classificar o perfil energético em categorias como Eficiente, Moderado ou Ineficiente, fornecendo recomendações e estimativas de impacto financeiro (tarifa de referência: R$ 0,75/kWh).

🏢 Setor

Sustentabilidade, energia e casas inteligentes — residências e pequenos estabelecimentos.

Stack Técnica

Python Scikit-Learn Random Forest Java Spring Boot OCI

📦 Entregas Obrigatórias (MVP)

  • Notebook com EDA e modelo supervisionado treinado
  • Endpoint POST /analise-energetica
  • Classificação de perfil energético com probabilidade
  • Recomendações de otimização energética
  • Estimativa de custo mensal (R$/mês)
  • Integração com ao menos um serviço OCI

⚡ Exemplo de Requisição

{
  "consumo_kwh": 420,
  "uso_horario_pico": true,
  "quantidade_equipamentos": 10,
  "tipo_imovel": "Casa",
  "horas_alto_consumo": 8
}

✅ Exemplo de Resposta

{
  "categoria": "Ineficiente",
  "probabilidade": 0.81,
  "custo_estimado_mensal": 315.00,
  "recomendacoes": [
    "Reduzir uso em horários
    de pico",
    "Avaliar aparelhos de
    alto consumo"
  ]
}

🚀 Recursos Opcionais

  • Dashboard de acompanhamento
  • Histórico de análises
  • Comparação entre períodos
  • Simulação de cenários de economia
  • Ranking de eficiência energética
OCI obrigatório: Object Storage · Compute · Functions

Tecnologia

Trilhas Técnicas

Cada equipe é composta por alunos de diferentes trilhas, colaborando para entregar uma solução completa.

🔬

Ciência de Dados

Exploração, limpeza e modelagem de dados com Python, Pandas e Scikit-Learn. Entrega de notebook com modelo treinado e serializado.

  • EDA e pré-processamento
  • Engenharia de atributos
  • Treinamento e avaliação
  • Serialização do modelo
⚙️

Back-End

Desenvolvimento de API REST com Java e Spring Boot, integrando o modelo de Data Science e expondo endpoints documentados.

  • API REST com Spring Boot
  • Integração com modelo DS
  • Validação e tratamento de erros
  • Documentação dos endpoints
☁️

Oracle Cloud (OCI)

Integração obrigatória com ao menos um serviço OCI para armazenamento, hospedagem ou processamento da solução.

  • Object Storage
  • OCI Compute
  • OCI Functions
  • Banco de dados (opcional)
